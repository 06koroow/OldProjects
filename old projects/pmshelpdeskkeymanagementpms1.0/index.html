Key Management (PMS)

Step-By-Step Procedure


   Before starting this process, you will need to have:
  • the Property's PID, Address
  • the Property's Address
    • or
  • The Host's Name
    Navigate to:Key Operations
  • While on this main screen, use the search bar to search by Tag ID for a key number that does not appear in the drop-down menu. If the number you type comes up in the dropdown below the search bar, it has already been assigned to a property.

  • In the top right-hand corner click "Add Keysets"
  • Search for the property using the "Property ID" or "Property Address" tabs, click the option in the dropdown to search


  • Enter the "Key Tag" number in the form as seen below:


  • Check the key-safe for a hook that is unused, enter the "Hook Number" in the corresponding text field

  • Enter the number of keys/fobs in the boxes provided

  • Click "add Set" to add an additional set of keys, click once per set of keys received.

  • Enter the number of keys and fobs on each set in the corresponding text boxes.

  • Click "Choose File" to upload a photo of the keysets, making sure any serial numbers, individual keys and fobs are all clearly visible.

  • Click "Submit"

  • Each keyset will now automatically be assigned a corresponding "Set Number", see the figure below for a what the finished page should show.

  • Write each "Tag Number" and "Set Number" on either side of a paper key-tag insert and attach to each set of keys

  • Add the keyset to the hook defined earlier in the sequence.

Step-By-Step Procedure

There are a number of circumstances where a property’s keysets will need to be edited, for example if a homeowner changes the locks or adds an alarm system to their property. The below procedure will use the following example (though the principles apply to the majority of cases where keys will need to be edited). “A homeowner has decided to change their locks, switching to a single security key and an alarm fob, rather than their previous three keys.” The keys have arrived at the Veeve office, so the new keys need to be swapped with the old keys and the old keys removed from PMS.

  1. Navigate to: Key Operations

  2. In the top right-hand corner click "Add Keysets"

  3. Search for the Property using either it’s Property ID or Address

  4. On the following screen (example shown below), edit the description, of keys, no. of fobs and image to reflect the new keys.

  5. Change the information to ensure it matches the master-set

  6. Press "Submit"

  7. Place the new keys on the corresponding sets

  8. Place the keysets back in the key safe and the remaining keysets in keyset.

Step-By-Step Procedure

This process can be completed for multiple keysets at once, but only one property at a time.

  1. Navigate to: Key Operations
  2. In the top right-hand corner click "Update Key Movement"

  3. On the following screen search for the Property using either it’s Property ID or Address


  4. On the following screen, you will be shown a list of keys and their curent "locations" as below:

  5. Keys will be listed in one of the following locations:
    • Key Safe- Within the Keysafe in the Veeve Office
    • Host- This set of keys has been returned to the Host and will need to be collected again before it can be used
    • Staff- This set of keys has been signed out and is with a member of staff, their name should be listed in the "Notes" section
    • Missing- This keyset is unable to be located by staff members, should this option be selected a member of staff must begin the "Lost Key Procedure"
    • In Property- This keyset is being left in the property, either for a guest stay or by host request
    • Concierge- This keyset is kept with the buildings concierge, please check the properties Ops Info for details
    • Keynest-This keyset is being stored in a keynest store, please see "Booking a Keynest Key" for further details
    • Booking- this keyset has been signed out for a booking
    • Not Working- This keyset has been found to not work as it should, this keyset should be replaced at the earliest possible point and the non functioning keys disposed of
    • Other- Any location/status that is not covered by the above categorie, please see the notes for details

  6. Select the keyset you wish to change the location of

  7. Using the dropdown at the bottom of the page, select which location the key has moved to


  8. At the bottom of the page, press the "Submit" button, and you will see a green bar at the top of the page, this means that you have succesfully updated the keyset's location.
