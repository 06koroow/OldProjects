Key Management (Keynest)

Step-By-Step Procedure


  • Navigate toKeynest

  • Log In with the Ops log-in details.

  • You will, by default be taken to “My Keys” where you will see a list of all Veeve keys on the platform.

  • Click “Add a new key” in the top right-hand corner of the screen


  • You will be asked to select whether the key is to be stored “In Office” or “In a Keynest Store”, for the purposes of this, please select “In a Keynest Store, further instructions will be provided for in office tagging at a later date.


  • You will be taken to a map view of your city, please select the Keynest you will be dropping the keys to. In selecting the Keynest, please bear in mind the following criteria:

    • Opening Hours – In selecting a Keynest, it is vital that the Keynest has opening hours that are useful to the business. Where possible opt for a Keynest with 24hr opening times or that is open between 6am and 10pm.

    • Distance- When selecting a Keynest, please ensure that it is at most 10 mins walking distance from either the property or a nearby transport hub.

  • You will then be taken to the below screen, where you will be asked to enter the key's name. The following format must be used: Key name - The tag ID followed by the set ID in brackets, with a space in the middle and no space at the end.


    • For example, if the tag ID is 012 and the set ID is (3456), the name would be formatted as below:



  • You will next be asked to select the “plan” the key will be used under, Veeve uses the "Monthly" Plan. Please select this option.

  • You will then be asked whether you wish to create a "Single Use" code, or a "Reoccurring" code, please only select single use.


  • You will then be given a "Collection Code” and a “Drop off Code”. The drop off code is to be given to the Keynest store so the key can be logged in-to the KN system.


  • The keys have now been added to Keynest and can be dropped off.

Step-By-Step Procedure

When booking keys for collection, the procedure begins in a similar manner to the process for adding keys.


  • Navigate to Keynest

  • Log In

  • You will, by default be taken to “My Keys” where you will see a list of all Veeve keys on the platform, in the search bar at the top, search for the set of keys you wish to book and click on the set.


  • Check the opening hours of the Keynest store to ensure the key will be able to be collected

  • Click “filter by key or address” and type in the tag and set ID’s in the same format as they were entered during key creation.

  • Scroll down till you find the key you require and click on it.

  • You will be brought to this page, showing the keys “status”. Check the opening hours of the Keynest Store are suitable for the proposed collection time, then click “get a collection Code”


  • As in the previous procedure, you will be brought to this page where you will specify which kind of code you require, and whether you want to restrict the validity period of the code generated to a certain period.


  • Once you have entered the above information click the “Book Now” button, to receive your code and schedule the key for collection.
